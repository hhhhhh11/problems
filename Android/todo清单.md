# 评估  
## Android应用开发 (30%)
### 常用知识 (50%)
熟悉四大组件、动画、布局、数据存储、常用控件、常用API等，完成应用开发；
初级：能够灵活使用。对四大组件声明周期，启动模式有清晰认知；常用组件 API 熟练使用；能熟练使用数据存储；对动画、布局性能有初步认知；按规范开发。
进阶：知晓原理。能根据交互需求选择合适方案或提供解决方案；能合理使用多线程并调优；对于常用 API 深入了解原理；能主动提炼封装工具类，并使用和优化调整工具类。
高级：能基于原理进一步扩展使用场景和创新。        3        
0~5
### 扩展知识 (30%)
熟悉H5交互、机型适配、NDK、自定义控件等，完成相关功能开发；
初级：能够灵活使用。能调用 so 库方法；有意识处理机型兼容、屏幕适配；能完成与 H5 交互；能熟练使用开源控件库；
进阶：知晓原理。对 View 事件分发有清晰认知；对 View 绘制流程有清晰了解；能选择合适项目的开源库并能调整；能完成 jni 层开发；合理完成机型兼容、屏幕适配；可以和 H5 完成特殊功能交互。
高级：能基于原理进一步扩展使用场景和创新。        2.5        
0~5
